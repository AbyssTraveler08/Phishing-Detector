# Phishing Link Detector

This project is a **Phishing Link Detector** built using **Python (Flask)** and **HTML/JavaScript** for the frontend. It detects phishing URLs using basic features like domain extraction and a simple heuristic.

## Features

- **Phishing Detection**: It checks whether a given URL is phishing or safe.
- **Frontend**: HTML form for users to input a URL and check its status.
- **Backend**: Flask-based Python app running the detection algorithm.
